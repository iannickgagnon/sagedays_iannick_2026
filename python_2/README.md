# Que contient de dossier?
Ce dossier contient le matériel utilisé lors de l'atelier **Python 2**:
1. `fichiers_csv_json_yaml/`: Artéfacts utilisés lors de la présentation. 
2. `presentation_x.pptx`: Diapositives utilisée lors de l'atelier (aussi en version `.pdf`).
3. `carnet_x.ipynb`: Carnet interactif qui suit la présentation.
4. `laboratoire_x.ipynb`: Carnet interactif **sans les solutions** du laboratoire.
5. `laboratoire_x_solution.ipynb`: Carnet interactif **avec les solutions** du laboratoire.

# Comment suivre?
* Les diapositives sont disponibles en format `.pdf` pour que vous puissiez suivre sans être encombré(e)s par les animations.
* Vous pouvez cliquer sur le bouton `Open in Colab` en haut à gauche des fichiers `.ipynb`. Cela créera une copie que vous pouvez modifier et exécuter dans votre espace **Google Drive**.
