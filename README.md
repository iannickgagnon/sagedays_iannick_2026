# Que contient ce dépôt?
Ce dépôt contient le matériel (présentations, carnets, ressources supplémentaires) de 3 ateliers animés par moi-même lors de l'événement **SageDays 2026** à l'ÉTS ([lien](https://www.etsmtl.ca/evenement/sage-days-1305)).

Il y a un dossier par atelier: 
* **Python 1** (`python_1/`)
* **Python 2** (`python_2/`)
* **Git 1** (`git_1/`)

Chaque dossier comprend minimalement:
1. `presentation_x.pptx`: Diapositives utilisée lors de l'atelier (aussi en version `.pdf`).
2. `carnet_x.ipynb`: Carnet interactif qui suivent la présentation.
3. `laboratoire_x.ipynb`: Carnet interactif **sans les solutions** du laboratoire.
4. `laboratoire_x_solution.ipynb`: Carnet interactif **avec les solutions** du laboratoire.
